19/***********************************************************
20
21 A Task is a called function which runs independently
22 of the rest of the kernel. When the task is started
23 (with KeStartTask), a little helper function gets
24 called, which calls the task's m_function.
25 When the m_function returns, the helper function
26 automatically kills the task.
27 (KeKillTask can also be used to kill a task prematurely)
28
29************************************************************/

93 The function needs to have a "void (int)" header.
